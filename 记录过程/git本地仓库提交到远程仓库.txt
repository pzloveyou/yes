首先在你的github上面创建一个仓库，然后复制你的仓库路径（等下有用）
    1.然后你的电脑上右键打开git base here
         过程如下：
         首先把远程仓库拉到本地，使用 （git clone 远程仓库路径）  的命令，这是你的本地就多了个
         文件了，然后修改完。
         就要提交到远程仓库
         先 git add  .   //本地仓库添加进来
         然后git commit -m “注释信息”
         然后 git push 远程仓库路径（分支：git push -u origin master //把本地代码推送到远程master分支上面）
   就可以了。
